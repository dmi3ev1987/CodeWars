Each number can be 0, 1, or 2 which represents the number of hands that person is holding up.</p><p>Now imagine there is a sequence which follows these rules:</p><ul><li>None of the people have their arms raised at first</li><li>Firstly, a person raises 1 hand; then they raise the second hand; after that they put both hands down - these steps form a cycle</li><li>Person #1 performs these steps all the time, person #2 advances only after person #1 puts their hands down, and person #3 advances only after person #2 puts their hands down</li></ul><p>The first 10 steps of the sequence represented as a table are:</p><pre><code>Step   P1   P2   P3-------------------- 0     0    0    0 1     1    0    0 2     2    0    0 3     0    1    0 4     1    1    0 5     2    1    0 6     0    2    0 7     1    2    0 8     2    2    0 9     0    0    1</code></pre><p>Given a number, return an array, containing 3 integers, each representing the number of hands raised by each person at that step, starting from <code>0</code>.</p>
