<h2><a href=https://www.codewars.com/kata/5840586b5225616069000001/train/python target="_blank">Which string is worth more?</a></h2><h3>7 kyu</h3><p>You will be given two ASCII strings, <code>a</code> and <code>b</code>. Your task is write a function to determine which one of these strings is "worth" more, and return it.</p><p>A string's worth is determined by the sum of its ASCII codepoint indexes. So, for example, the string <code>HELLO</code> has a value of 372: H is codepoint 72, E 69, L 76, and O is 79. The sum of these values is 372.</p><p>In the event of a tie, you should return the first string, i.e. <code>a</code>.</p>
