The first playable version was completed on June 6, 1984. Pajitnov derived its name from combining the Greek numerical prefix tetra- (the falling pieces contain 4 segments) and tennis, Pajitnov's favorite sport.</p><h1 id="about-scoring-system">About scoring system</h1><p>The scoring formula is built on the idea that more difficult line clears should be awarded more points. For example, a single line clear is worth <code>40</code> points, clearing four lines at once (known as a Tetris) is worth <code>1200</code>.</p><p>A level multiplier is also used. The game starts at level <code>0</code>. The level increases every ten lines you clear. Note that after increasing the level, the total number of cleared lines is not reset.</p><p>For our task you can use this table:</p><table class="demo">    <tbody><tr>        <th>Level</th>        <th>Points for 1 line<br></th>        <th>Points for 2 lines</th>        <th>Points for 3 lines</th>        <th>Points for 4 lines</th>    </tr>    <tr>        <td>0</td>        <td>40</td>        <td>100<br></td>        <td>300</td>        <td>1200</td>    </tr>    <tr>        <td>1</td>        <td>80</td>        <td>200</td>        <td>600</td>        <td>2400</td>    </tr>    <tr>        <td>2</td>        <td>120</td>        <td>300</td>        <td>900</td>        <td>3600</td>    </tr>    <tr>        <td>3</td>        <td>160</td>        <td>400</td>        <td>1200</td>        <td>4800</td>    </tr>    <tr>        <td>...</td>    </tr>    <tr>        <td>7</td>        <td>320</td>        <td>800</td>        <td>2400</td>        <td>9600</td>    </tr>    <tr>        <td>...</td>        <td colspan="4">For level n you must determine the formula by yourself using given examples from the table.</td>    </tr></tbody></table><h1 id="task">Task</h1><p>Calculate the final score of the game using original Nintendo scoring system</p><h1 id="input">Input</h1><p>Array with cleaned lines.<br>Example: <code>[4, 2, 2, 3, 3, 4, 2]</code><br>Input will always be valid: array of random length (from <code>0</code> to <code>5000</code>) with numbers from <code>0</code> to <code>4</code>.</p><h1 id="ouput">Ouput</h1><p>Calculated final score.<br><code>def get_score(arr) -&gt;
