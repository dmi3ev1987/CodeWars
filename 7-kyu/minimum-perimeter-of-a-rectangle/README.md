<h2><a href=https://www.codewars.com/kata/5826f54cc60c7e5266000baf/train/python target="_blank">Minimum Perimeter of a Rectangle</a></h2><h3>7 kyu</h3><p>A rectangle can be defined by two factors: height and width.</p><p>Its area is defined as the multiplication of the two: height * width.</p><p>Its perimeter is the sum of its four edges: height + height + width + width.</p><p>It is possible to create rectangles of the same area but different perimeters.For example, given an area of 45, the possible heights, widths and resultant perimeters would be:</p><p>(1, 45) = 92</p><p>(9, 5) = 28</p><p>(15, 3) = 36</p><p>Note that (6, 7.5) has an area of 45 too, but is discarded in this kata because its width is non integral.</p><p>The task is to write a function that, given an area as a positive integer, returns the smallest perimeter possible of a rectangle with integral side lengths.</p><h3 id="input-range">Input range:</h3><p>1 &lt;= area &lt;= 5 x 10 ^ 10</p>
