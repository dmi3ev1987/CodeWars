<h2><a href=https://www.codewars.com/kata/56f399b59821793533000683/train/python target="_blank">Sort deck of cards</a></h2><h3>7 kyu</h3><p>Write a function <code>sort_cards()</code> that sorts a shuffled list of cards, so that any given list of cards is sorted by rank, no matter the starting collection.</p><p>All cards in the list are represented as strings, so that sorted list of cards looks like this:</p><p><code>['A', '2', '3', '4', '5', '6', '7', '8', '9', 'T', 'J', 'Q', 'K']</code></p><p>Example:</p><pre><code>&gt;&gt;&gt; sort_cards(['3', '9', 'A', '5', 'T', '8', '2', '4', 'Q', '7', 'J', '6', 'K'])['A', '2', '3', '4', '5', '6', '7', '8', '9', 'T', 'J', 'Q', 'K']</code></pre><p>Hint: <em>Tests will have many occurrences of same rank cards, as well as vary in length. You can assume though, that input list is always going to have at least 1 element.</em></p>
