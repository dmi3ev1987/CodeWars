<h2><a href=https://www.codewars.com/kata/54e2213f13d73eb9de0006d2/train/python target="_blank">Pair Zeros</a></h2><h3>7 kyu</h3><h3 id="task">Task</h3><p>For a given list of digits <code>0</code> to <code>9</code>, return a list with the same digits in the same order, but with all <code>0</code>s paired. Pairing two <code>0</code>s generates one <code>0</code> at the location of the first one.</p><h3 id="examples">Examples</h3><pre><code>input: [0, 1, 0, 2]paired: ^-----^    -&gt; [0, 1,   2]  kept: ^input: [0, 1, 0, 0]paired: ^-----^    -&gt; [0, 1,    0]  kept: ^        ^input: [1, 0, 7, 0, 1]paired:    ^-----^    -&gt; [1, 0, 7,    1]  kept:    ^input: [0, 1, 7, 0, 2, 2, 0, 0, 1, 0]paired: ^--------^        ^--^    -&gt; [0, 1, 7,    2, 2, 0,    1, 0]  kept: ^                 ^        ^</code></pre><h3 id="notes">Notes</h3><ol><li>Pairing happens from left to right. For each pairing, the second <code>0</code> will always be paired towards the first ( right to left )</li><li><code>0</code>s generated by pairing can NOT be paired again</li><li>( void where not applicable: ) Don't modify the input array or you may fail to pass the tests</li></ol>
